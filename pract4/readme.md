### 1. Моделювання енергетичних потоків
**Опис:** Реалізуйте симуляцію потоків енергії в мережі за допомогою диференціальних рівнянь та бібліотеки SciPy. Візуалізуйте потоки енергії за часом, використовуючи графіки для порівняння ефективності методів.
**Пояснення коду:** Код використовує функцію `odeint` з бібліотеки `scipy.integrate` для розв'язання простого звичайного диференціального рівняння (ЗДР), яке моделює експоненційне загасання енергетичного потоку ($dy/dt = -ky$). Створюється часовий масив (від 0 до 20), і обчислюється динаміка потоку. Для порівняння генерується масив статичного середнього значення (константа 50). За допомогою `matplotlib` обидва масиви виводяться на один графік для візуального порівняння динамічної моделі та статичного підходу.

### 2. Виявлення змін у трендах
**Опис:** Використовуючи метод CUSUM, визначте моменти різких змін у тренді споживання енергії. Створіть графік для порівняння цих змін із трендами.
**Пояснення коду:**
Генерується масив зі 100 випадкових значень із нормальним розподілом. Щоб зімітувати різку зміну тренду, до другої половини даних (індекси від 50 і далі) додається константа 5. Алгоритм CUSUM реалізується шляхом обчислення кумулятивної суми відхилень кожного значення від загального середнього (`np.cumsum(data - np.mean(data))`). На графіку відображаються як вихідні дані (де зміна ледь помітна візуально через шум), так і лінія CUSUM, яка чітко показує точку зламу тренду.

### 3. Прогнозування за допомогою LSTM

Модель `Sequential` складається з одного шару `LSTM` (10 нейронів) та вихідного повнозв'язного шару `Dense`. Модель компілюється з оптимізатором Adam та функцією втрат MSE, навчається 2 епохи (для демонстрації), після чого робить передбачення. На графіку порівнюються справжні значення (`y`) та передбачені (`pred`).

На діаграмі розсіювання точки забарвлюються відповідно до передбачень моделі, візуально виділяючи знайдений викид.
