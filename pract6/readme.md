# Практична робота 6: Аналіз впливу побутових пристроїв на магнітне поле
**Виконав: Михайлець Артем ТВ-33**
**Варіант 13**

## 📌 Опис завдання
Метою роботи є збір реальних даних за допомогою мобільного додатка (Phyphox), їх обробка мовою Python та аналіз електромагнітного фону в різних зонах житлового приміщення та поблизу промислових об'єктів (підстанцій).

## 🧠 Логіка обробки даних
1. **Обчислення модуля індукції:** Оскільки датчик телефону видає значення за трьома осями ($X, Y, Z$), програма розраховує повний вектор магнітної індукції за формулою: $B = \sqrt{B_x^2 + B_y^2 + B_z^2}$.
2. **Фільтрація та пошук аномалій:** Використано метод **Z-Score**. Значення, що відхиляються від середнього більше ніж на 3 стандартні відхилення, класифікуються як аномальні сплески (моменти вмикання приладів).
3. **Порівняльний аналіз:** Статистичне порівняння середніх рівнів випромінювання між спальнею, кухнею та вулицею.
